User Guideline

  • 주어진 지문을 다른 언어로 번역하는 작업입니다.
  • Instruction에 출발 언어와 도착 언어를 명시해 주세요.
  • Input에 출발 언어의 텍스트를 작성해 주세요.
  • Output에 도착 언어의 번역문을 작성해 주세요. 번역문은 Input의 텍스트와 같은 의미의 텍스트여야 합니다.

Prompt Example

Instruction

  • 다음 영어 문장을 한글로 번역해 주세요.

Input

  • I overslept today.

Output

  • 나는 오늘 늦잠을 잤다.
