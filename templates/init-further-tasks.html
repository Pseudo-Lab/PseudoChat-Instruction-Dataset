Further Tasks

  • Description Task : 주어진 개념에 대해 설명하는 작업
  • Entailment Task : 두 문장이 연결되는지에 대한 작업
  • (Multiple / Binary) Choice task : 주어진 선택지 내에서 정답을 고르는 작업
  • Calculation Task : 주어진 지문에서 정량 데이터를 뽑아내 계산하는 작업
  • Grammar Check Task : 주어진 지문이 문법적으로 올바른지 검사 또는 수정하는 작업
  • Code Generation Task : 요구 사항에 맞는 코드를 작성하는 작업
  • 그 외 다양한 Generation Tasks...

Thank you for contributing to PseudoChat training data!
